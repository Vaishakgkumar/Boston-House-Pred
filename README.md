In This machine learning project, Boston House Prediction, I built a model that predicts the prices of houses in Boston based on various features. I trained the model using historical dataset provided by **sklearn** and saved it using **pickle**. I then used the Flask web framework to create a frontend for the project and deployed it on the Python Anywhere platform. This project showcases my skills in machine learning, web development, and cloud computing, and demonstrates my ability to work independently on a complex project.
